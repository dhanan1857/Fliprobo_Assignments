11.	What is the main difference between a web scraper and a web crawler?

Ans - Web Scraper-  Extracting data from websites in an automated manner.It fetches data information in precise 
      and accurte manner.
      Web Scraper - Indexing the information on the page using bots also known as crawlers.Web crawlers or
      navigate through heaps of data and information and procure whatever is relevant for your project
      
12. What is ‘robots.txt’ file? What is the use of ‘robots.txt’ file?

Ans - A robots. txt file is a simple text file which webmasters can create to tell web crawlers which parts of
      a website should be crawled and which should not.
      
13. What are static and dynamic web pages

Ans - Static web pages are static in nature means it contain the same prebuilt content each time the page is
      loaded.
      Dynamic Web pages are dynamic in nature which means it displays different informationeah time page is
      loaded
